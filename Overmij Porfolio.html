Wie ben ik?

Wat doe ik?

Op het ROC Nijmegen werk ik aan verschillende software- en webprojecten. Ik bouw webpagina’s met HTML en CSS, schrijf code in Java en werk aan opdrachten waarin ik complete onderdelen van applicaties ontwikkel.

Daarnaast werk ik samen met klasgenoten aan projecten, denk ik mee over oplossingen en test ik de functionaliteit van onze programma’s. Ook documenteer ik mijn werk en zorg ik dat mijn code duidelijk en netjes blijft voor het projectteam.

Over mij

Ik ben Tygo Diks, 17 jaar oud, en ik volg de opleiding Software Developer op het ROC in Nijmegen. Ik werk daar aan verschillende projecten waarin ik websites en software bouw, en ik vind het leuk om mezelf steeds verder te ontwikkelen in programmeren.

Buiten school houd ik me graag bezig met sport, want ik zit op voetbal en ga ook naar de sportschool. In mijn vrije tijd game ik vaak en breng ik veel tijd door met vrienden. Daarnaast ben ik een grote Formule 1-fan, ik volg de races graag en ben al naar een paar races geweest.

Mijn Ervaring

Ik werk nu al bijna 3 jaar bij de albert heijn. Ik werk daar als vakkenvuller met wat vrienden van mij. Ik vind het daar leuk om te werken alleen kijk wel naar mogelijkheden om te vertrekken.
